### linux 系统备份是怎么备份的
系统的备份的命令
```
cd /
sudo tar cvpzf backup.tgz --exclude=/proc --exclude=/mnt --exclude=/sys --exclude=/backup.tgz /

```
系统的还原
```
tar xvpfz backup.tgz -C /
```
